    크롬 브라우저를 열어주세요
    바탕화면이나 작업표시줄(화면 맨 아래)에서 크롬(Chrome) 또는 엣지(Edge) 아이콘을 클릭해서 브라우저를 엽니다.

    https://github.com/new
  • 3
    로그인 화면이 나오면 로그인하세요
    Username: dhkks5908-design 과 비밀번호를 입력하고 [Sign in] 버튼을 클릭합니다.

  • 4
    "Create a new repository" 화면이 나옵니다
    아래처럼 생긴 화면이 나타납니다. Repository name 이라고 쓰인 입력칸을 찾아주세요.

  • 5
    Repository name 칸에 정확히 입력하세요 ⭐ 가장 중요!

index.html 파일 올리기

다운받은 파일을 GitHub에 드래그해서 올리는 단계

  • 1
    저장소가 만들어졌습니다! 이런 화면이 보여요
    아래처럼 생긴 화면이 나타납니다. 파란 글씨 "uploading an existing file" 을 찾아서 클릭하세요!
    github.com/dhkks5908-design/dhkks5908-design.github.io
    Quick setup — if you've done this kind of thing before Get started by creating a new file or uploading an existing file ← 클릭! We recommend every repository include a README, LICENSE, and .gitignore.
    💡
    "uploading an existing file" 이 안 보인다면?화면 위쪽에 있는 Add file 버튼 클릭 → Upload files 클릭하세요!
  • 2
    파일 업로드 화면이 나타납니다
    큰 점선 박스가 있는 화면이 나옵니다. 이 박스 안에 파일을 끌어다 놓을 거예요.
    ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐ │ │ │ Drag files here to add them │ │ ↑ 파일을 여기로 끌어다 놓으세요! │ │ │ │ or choose your files │ └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘
  • 3
    index.html 파일 찾기
    파일 탐색기를 열어서 파일을 찾아야 합니다. 아래 순서대로 하세요:
    📁 파일 찾는 방법
    방법 1 — 작업표시줄에서 열기:
    화면 맨 아래 작업표시줄에서 노란 폴더 아이콘(📁) 클릭

    방법 2 — 단축키:
    키보드에서 윈도우키(⊞) + E 동시에 누르기

    파일 탐색기 열리면:
    왼쪽 메뉴에서 "다운로드" 클릭 → index.html 파일 찾기
  • 4
    파일을 드래그 앤 드롭으로 올리기!
    ① 파일 탐색기와 브라우저를 화면에 나란히 띄워두세요
    ② index.html 파일을 마우스 왼쪽 버튼으로 꾹 누른 채로
    ③ 브라우저의 점선 박스 안으로 끌어다 놓기!
    ✅
    업로드 성공하면?파일 이름 옆에 초록 체크(✓)가 생깁니다. 그러면 성공!
    💡
    드래그가 어렵다면?점선 박스 안의 choose your files 파란 글씨를 클릭 → 파일 탐색기가 열리면 index.html 선택 → 열기 버튼 클릭!
  • 5
    아래로 스크롤 → Commit changes 버튼 클릭!
    파일이 올라간 후 페이지를 아래로 스크롤하면 저장 버튼이 나옵니다.

기다렸다가 사이트 확인하기!

GitHub가 자동으로 사이트를 만들어 줍니다. 1~3분이면 됩니다!

  • 1
    1~3분 기다리세요 ☕
    Commit changes 버튼을 누른 후 GitHub가 자동으로 사이트를 만들어 줍니다. 아무것도 안 해도 됩니다! 잠깐 기다리세요.
  • 2
    배포 완료 확인하는 방법 (선택사항)
    빨리 확인하고 싶다면? 저장소 페이지에서 [Actions] 탭을 클릭하면 배포 진행 상태를 볼 수 있어요.
  • 3
    내 사이트 접속하기! 🎉
    브라우저 주소창에 아래 주소를 입력하고 Enter를 누르세요!
    https://dhkks5908-design.github.io
    🎉
    사이트가 열리면 배포 성공!이 주소를 카카오톡, 인스타, 문자로 공유하면 전 세계 누구나 볼 수 있습니다!
    😰
    404 오류가 뜬다면?
    • 아직 1~2분이 안 지났을 수 있어요 → 더 기다리기
    • Ctrl+Shift+R 로 강력 새로고침 해보기
    • 저장소 이름이 dhkks5908-design.github.io 인지 확인
    • 파일 이름이 index.html (소문자) 인지 확인

PART 2 — 사이트 수정하기

배포 이후 내용을 바꾸고 싶을 때! 두 가지 방법 중 편한 걸 선택하세요.

🤔
어떤 방법을 쓸까요?
  • 방법 A (추천) — GitHub 웹사이트에서 바로 수정. 설치 없이 간단!
  • 방법 B — 새 index.html 파일을 만들어서 덮어씌우기. 큰 수정할 때 편함!
A

방법 A — GitHub 웹에서 직접 수정 ✅ 추천

브라우저에서 바로 파일 내용을 수정할 수 있어요. 가장 간단!

  • 1
    내 저장소 페이지로 이동
    아래 주소를 복사해서 브라우저 주소창에 붙여넣기 후 Enter!
    https://github.com/dhkks5908-design/dhkks5908-design.github.io
  • 2
    파일 목록에서 index.html 클릭
    저장소 페이지에 파일 목록이 보입니다. 거기서 index.html 파란 글씨를 클릭하세요.
  • 3
    연필 아이콘(✏️) 클릭 — 수정 모드 시작
    파일 내용이 보이면, 오른쪽 위를 보면 연필 모양 아이콘이 있습니다. 그걸 클릭하면 수정 모드가 됩니다.
  • 4
    원하는 내용을 직접 수정하세요
    파일 내용이 수정 가능한 상태가 됩니다. 찾고 싶은 내용은 Ctrl+F 로 검색할 수 있어요!
  • 5
    오른쪽 위 초록 버튼 클릭!
    수정이 끝나면 오른쪽 위에 있는 초록 버튼을 클릭합니다.
  • 6
    팝업이 뜹니다 → 초록 버튼 한 번 더 클릭!
    작은 창이 뜹니다. 아무것도 바꾸지 말고 바로 초록 버튼을 클릭합니다.
  • 7
    1~2분 후 사이트에서 확인!
    아래 주소를 열고 변경된 내용이 보이는지 확인하세요.
    https://dhkks5908-design.github.io
B

방법 B — 새 파일 올려서 교체하기

수정된 index.html 파일을 새로 올리면 자동으로 덮어씁니다.

  • 1
    저장소 페이지로 이동
    https://github.com/dhkks5908-design/dhkks5908-design.github.io
  • 2
    [Add file] → [Upload files] 클릭
    페이지 상단의 Add file 버튼을 클릭 → 드롭다운에서 Upload files 클릭
  • 3
    수정된 index.html 파일을 올리기
    새로 만든 또는 수정된 index.html 파일을 드래그 앤 드롭으로 올립니다.
    파일 이름이 같으면(index.html) 자동으로 덮어씁니다!
  • 4
    [Commit changes] 초록 버튼 클릭 → 완료!
    1~2분 후 사이트에서 바뀐 내용을 확인하세요.

자주 막히는 문제 & 해결법

막혔을 때 당황하지 말고 여기서 찾아보세요!

  • Q
    사이트 접속하면 404 오류가 떠요
    🔧
    해결 순서대로 해보세요:
    • 아직 시간이 덜 됐을 수 있어요 → 3~5분 더 기다린 후 재접속
    • Ctrl+Shift+R 로 강력 새로고침
    • 파일 이름 확인: 반드시 소문자 index.html
    • 저장소 이름 확인: dhkks5908-design.github.io
  • Q
    수정했는데 사이트에 반영이 안 돼요
    🔧
    해결 방법:
    • Ctrl+Shift+R (강력 새로고침) 눌러보기
    • 시크릿 창(Ctrl+Shift+N) 에서 접속해보기
    • 1~2분 더 기다렸다가 재접속
  • Q
    로그인을 못 하겠어요 / 비밀번호를 잊었어요
    🔧
    해결 방법:
    • github.com 로그인 화면에서 Forgot password? 클릭
    • 가입할 때 쓴 이메일 주소 입력 → 이메일로 재설정 링크 발송
