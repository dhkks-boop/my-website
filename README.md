# My Website

This repository contains a simple static website built with HTML and published with GitHub Pages.

Live site:

- https://dhkks-boop.github.io/my-website/

## Files

- `index.html`: main website page

## How to update the site

After editing `index.html`, run:

```powershell
git add .
git commit -m "update website"
git push
```

GitHub Pages will automatically redeploy the site after the push.
