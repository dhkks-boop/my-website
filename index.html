🖥️ 컴퓨터 기본 시작

각 부품의 이름과 역할 · 켜고 끄는 올바른 순서

1
컴퓨터 부품 이름 알기
📖 개념
컴퓨터는 여러 부품이 함께 일하는 기계예요. 각 부품이 무엇을 하는지 먼저 알아봅시다!
🖥️
모니터
화면을 보여주는 장치. TV처럼 글자·사진·영상이 표시됩니다.
🗄️
본체
컴퓨터의 두뇌. 모든 계산이 여기서 일어납니다.
⌨️
키보드
글자·숫자·명령을 입력하는 장치.
🖱️
마우스
화면의 화살표를 움직여 원하는 곳을 클릭합니다.
🔊
스피커
음악·영상 소리를 출력하는 장치.
📡
공유기
인터넷 신호를 집 안에 무선으로 퍼뜨립니다.
💡 노트북은 이 모든 부품이 하나로 합쳐진 것!

노트북은 본체+모니터+키보드+마우스가 한 몸처럼 붙어 있습니다.

2
컴퓨터 켜는 순서 — 직접 해보기
🖱️ 실습
아래 버튼을 순서대로 눌러보세요. 컴퓨터 켜는 순서를 연습합니다!
▼ 순서대로 클릭해보세요
① 모니터부터 먼저 켜세요!
3
컴퓨터 올바르게 끄기
📖 개념
컴퓨터를 끌 때는 반드시 시작 버튼 → 전원 → 시스템 종료 순서로 해야 합니다.
방법언제 쓰나요?특징
🔴 시스템 종료오늘 더 안 쓸 때완전히 꺼짐
🔁 재시작오류 해결 후껐다가 다시 켜짐
💤 절전 모드잠깐 자리 비울 때빠르게 다시 사용 가능
💡 종료 방법

왼쪽 아래 윈도우 버튼 → 전원 → 시스템 종료를 선택하세요.

🎯 1단원 확인 퀴즈

배운 내용을 확인해봅시다!

Q1. 컴퓨터를 켤 때 올바른 순서는?
✅ 정답! 모니터를 먼저 켜야 본체 화면이 제대로 표시됩니다.
❌ 틀렸어요. 모니터를 먼저 켜고, 그 다음 본체를 켜야 합니다.
Q2. 컴퓨터를 끌 때 올바른 방법은?
✅ 정답! 정상 종료가 가장 안전합니다.
❌ 틀렸어요. 코드 뽑기와 강제 종료는 위험합니다.
